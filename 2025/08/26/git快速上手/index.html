git快速上手

发表于 2025-08-26 分类于 git 本文字数： 575 阅读时长 ≈ 1 分钟

要在本地创建Git仓库仓库并连接到远程仓库 git@github.com:xxxx/xxxxxxxx.git，可以按照以下步骤操作：

1.首先在本地选择一个目录作为仓库存放位置，打开终端并进入该目录：

cd /path/to/your/local/directory

2.初始化本地Git仓库：

git init

3.将本地仓库与远程仓库关联：

git remote add origin git@github.com:xxxxx/xxxxxx.git

验证远程仓库连接是否成功：

git remote -v

如果成功，会显示类似以下内容：

从远程仓库拉取代码（如果远程已有内容）：

git pull origin main

(如果默认分支不是main，可以替换为实际的分支名，如master）

4.添加到本地工作区：

git add .

5.提交给本地仓库：

git commit -m "这是提交的描述内容"

6.提交到远程仓库：

git push origin master (master对应提交的远程仓库分支）

创建并切换本地仓库分支：
git checkout -b dev

提交当前分支：

git push -u origin dev
