一、环境准备

  1. 安装 Node.js

    • 访问 Node.js 官网 下载并安装（推荐 LTS 版本）
    • 验证安装：node -v 和 npm -v 能显示版本号

  2. 安装 Git

    • 访问 Git 官网 下载对应系统版本
    • 验证安装：git --version

    验证安装：hexo -v
